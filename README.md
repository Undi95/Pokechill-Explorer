## ✨ Fonctionnalités

### 🔍 Exploration Complète
- **Pokédex interactif** avec recherche avancée (types, stats, talents, egg moves)
- **Détails complets** : BST, stats réelles, talents cachés, signatures
- **Divisions** (SSS, SS, S, A, B, C, D) pour évaluer rapidement la puissance

### 🛠️ Team Builder
- **Création d'équipes** de 6 Pokémon avec sauvegarde locale
- **Personnalisation avancée** :
  - ⭐ **IVs** (0-6 étoiles) configurables par stat
  - 🌿 **Natures** (6 disponibles) avec coloration des stats boostées/baissées
  - ✨ **Star Signs** (Sol, Luna, Pluto, Ceres, Terra, Eris) pour les shiny
  - 🎗️ **Médailles** (ribbons) visibles sur les cartes
- **Export GIF animé** de vos teams en HD ou miniature

### ⚔️ Calculateur de Dégâts
- **Simulation de combats** en mode Wild ou Raid
- **Prise en compte** :
  - Natures de l'attaquant
  - Talents et objets
  - Buffs de cuisine
  - Types et stabilités
- **Affichage des stats** colorées selon la nature

### 🌟 Featured Teams
- **Teams communautaires** partagées et archivées
- **Manifest auto-généré** pour faciliter les contributions
- **Système de tags** par version et auteur

---

## 🚀 Technologies

- **Frontend** : Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Hébergement** : GitHub Pages
- **Données** : Parsing automatique des fichiers du jeu PokeChill
- **Cache** : IndexedDB pour les performances
- **Export** : Bibliothèque GIF.js pour les animations
